# Archivos de Audio

Esta carpeta contiene los archivos de sonido utilizados en la terminal:

- `key.mp3`: Sonido de tecleo
- `enter.mp3`: Sonido al presionar Enter
- `error.mp3`: Sonido de error

Nota: Debes agregar estos archivos manualmente o eliminar las referencias a ellos en el código si no los tienes disponibles.
